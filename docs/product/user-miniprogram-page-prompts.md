# 用户端小程序单页效果图提示词

用途：逐页生成「悦享e食」用户端微信小程序 UI 效果图。生成后按下方文件名放入 `docs/product/user-miniprogram-reference/`，后续前端按图片一比一还原。

统一要求：

- 每张图只生成一个页面，不要三联屏、海报、手机外框、透视机身、营销背景或额外标题。
- 画布为竖屏小程序截图，建议 `1080x1920` 或 `9:16`，正视角，完整页面从顶部导航到安全区底部。
- 风格是高质量原生微信小程序 UI，主色 `#009bf5`，背景 `#f3f6f9`，卡片白色，正文深灰黑，辅助文字灰色。
- 后续新生成或优化的子页面默认使用白色原生导航栏，黑色标题和返回箭头；不要再使用整条蓝色顶部栏，除非用户单独指定。
- 视觉参考：美团/饿了么级别的信息密度和商业质感，但不要出现它们的品牌、logo 或完全复制它们的设计。
- 使用圆角卡片、清晰分组、真实食品缩略图、清爽图标、底部固定操作栏；文字必须是清晰可读的中文。
- 避免：英文乱码、错别字、过多渐变、深色主题、3D 装饰、线框图、SVG 风格、代码、网页浏览器外壳。
