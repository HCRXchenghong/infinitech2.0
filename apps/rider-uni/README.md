# 骑手端 uni-app

首期覆盖在线/离线、抢单、派单、每日一次免责取消、任务流转、钱包和客服。

当前已落地：

- 抢单大厅首屏：上线/离线、可接容量、距离上报、保证金缴纳/微信免押/退押申请、确认取货、送达完成、每日一次免责取消、拒绝当前派单，完成固定单量后的拒派会提示本次免责。
- 站长工作台首屏：站点骑手列表、骑手等级/接单耗时、待调度订单列表、手动派单、每日任务时长和固定单量配置。
- 邀约入口：站长可创建本站骑手邀约，管理员可创建骑手/站长邀约，接受邀约后签发骑手端登录 token。
- 默认访问 BFF：`http://127.0.0.1:25500`，本地 token 可在 storage 中配置 `riderToken` 和 `stationManagerToken`。

账号与调度规则：

- 骑手端分为站长账号和骑手账号。
- 站长和骑手都采用邀约制注册，不能公开自助注册。
- 站长可以查看站点全部数据、配置每日任务时长并手动派单。
- 下单 10 分钟内进入抢单大厅，10 分钟及以上未抢单则自动派单。
- 骑手拒绝派单后，系统派给下一位符合条件的在线骑手。
- 自动派单按骑手等级优先级、平均接单耗时、距离和在线负载排序。
- 站长和后台可查看骑手接单时间、平均接单耗时、日均完成单数、完成率、站点团队均值和等级。
- 后台配置每日固定订单数；骑手完成固定数后，后续派单可免责不接并留审计。
- 骑手缴纳 50 元保证金或微信免押审核通过后才能接单。
- 骑手退保证金从最后一单完成时间/提交离职时间中较晚者起算一周；有纠纷则从纠纷截止后顺延一周。
- 骑手可在私聊或群聊中接收/发送余额红包，资金和领取记录进入钱包流水。
- 旧版已有能力必须保留：账单明细、钱包充值/提现、收入明细、数据统计、头像上传、个人信息、修改手机号、修改密码、接单设置、健康证、保险保障、违规申诉、骑手之家、历史订单、客服。
